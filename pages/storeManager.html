Store Manager

Projeto desenvolvido pelo curso de Desenvolvimento Web da Trybe no qual foi construída aplicação uma API utilizando a arquitetura MSC (model-service-controller). A API construída é um sistema de gerenciamento de vendas no formato dropshipping em que é possível criar, visualizar, deletar e atualizar produtos e vendas. Foi utilizado o banco de dados MySQL para a gestão de dados. Além disso,trata-se de uma API RESTful.

Nesse projeto eu desenvolvi as seguintes skills:
  • Conectei a aplicação com um banco de dados MySQL;
  • Exercitei todo processo de CRUD;
  • Estruturei a aplicação nas camadasMSC (model-service-controller);
  • Deleguei as responsabilidades para cada camada;
  • Apliqui os padrões REST;
  • Escrevi testes unitários para a aplicação;
