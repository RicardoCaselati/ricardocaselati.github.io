Talker Manager

Projeto desenvolvido pelo curso de Desenvolvimento Web da Trybe no qual foi construída aplicação de cadastro de talkers (palestrantes) em que será possível cadastrar, visualizar, pesquisar, editar e excluir informações.

Nesse projeto eu desenvolvi as seguintes skills:
  • Desenvolvi uma API de um CRUD (Create, Read, Update e Delete) de palestrantes (talkers);
  • Desenvolvi alguns endpoints que irão ler e escrever em um arquivo utilizando o módulo fs;
