Trybewarts

Projeto desenvolvido pelo curso de Desenvolvimento Web da Trybe no qual foi criada uma página de formulário da Escola de Magia de Trybewarts, em que as pessoas estudantes poderão enviar seus feedbacks sobre ela. O tema desse projeto é baseado na obra 'Harry Potter', de J. K. Rowling!

Nesse projeto eu desenvolvi as seguintes skills:
  • Formulários em HTML;
  • CSS Flexbox para criar layouts flexíveis;
  • Regras CSS específicas para serem aplicadas a dispositivos móveis;
  • Páginas que alteram o seu layout de acordo com a orientação da tela;
  • Estilização feita em bootstrap, HTML e CSS.
